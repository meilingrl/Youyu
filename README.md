# CampusMarket

`CampusMarket` is the main project workspace for the course repository.

It contains the real product, documentation, code, scripts, and test assets for the campus marketplace system.

## Top-Level Structure

- `docs/` - structured project documentation
- `frontend/` - Vue 3 frontend
- `backend/` - Spring Boot backend
- `database/` - database design assets and migration references
- `resources/` - supporting reference assets
- `scripts/` - helper scripts
- `tests/` - test assets and records
