# 功能需求

## 需求分级说明

- 必须支持：当前阶段直接影响系统成立与验收
- 应支持：高优先级增强，允许在主链稳定后补齐
- 可选增强：后续迭代空间

## 1. 前台用户端

### 1.1 账号与认证

必须支持：

- 学生用户注册
- 学生用户登录
- 退出登录
- 学生身份认证申请
- 认证状态查看
- 认证前后权限差异控制
- 一学号一账号约束

应支持：

- 校园邮箱辅助校验
- 认证失败原因反馈
- 人工审核流转
- 异常账号二次核验

### 1.2 首页与商品浏览

必须支持：

- 首页推荐区
- 分类导航
- 搜索入口
- 热门商品展示
- 最新上架展示
- 商品列表
- 分类筛选
- 关键字搜索
- 排序
- 商品详情页

应支持：

- 按专业、学龄、场景筛选
- 店铺入口

### 1.3 收藏与浏览

必须支持：

- 收藏商品
- 取消收藏
- 查看收藏列表

应支持：

- 浏览历史记录
- 最近浏览摘要

### 1.4 购物车与订单

必须支持：

- 加入购物车
- 修改数量
- 删除商品
- 订单预览
- 提交订单
- 在线模拟支付
- 我的订单
- 取消订单
- 确认收货
- 退款申请基础能力

当前必须支持的交付方式：

- 校内线下交付
- 物流寄送
- 电子交付

电子交付当前必须支持：

- 局部预览
- 确认收货后开放完整下载
- 下载留痕
- 不支持退货
- 举报入口

### 1.5 用户中心

必须支持：

- 个人信息查看
- 个人信息修改
- 地址管理
- 我的发布
- 我的订单
- 我的收藏
- 我的认证状态
- 我的信用摘要

应支持：

- 个人统计摘要
- 浏览记录摘要
- 默认设置入口

## 2. 卖家与店铺能力

### 2.1 商品发布

必须支持：

- 发布商品
- 编辑商品
- 下架商品
- 删除商品
- 查看发布记录
- 配置可用交付方式

资料类商品必须支持：

- 提交后进入待审核
- 管理员审核通过后上架
- 驳回原因反馈

### 2.2 店铺能力

必须支持：

- 店铺申请
- 店铺审核或开通流程
- 店铺主页
- 店铺名称与简介
- 店铺商品列表

应支持：

- 店铺公告
- 店铺标签
- 店铺评分展示
- 常客优惠预留

### 2.3 信用展示

必须支持：

- 认证状态展示
- 好评率展示
- 交易完成数展示
- 违规或限制状态提示

应支持：

- 店铺信用等级
- 响应速度展示

## 3. 后台管理端

### 3.1 管理员认证

必须支持：

- 管理员登录
- 登录态校验
- 未登录访问拦截

### 3.2 用户与认证管理

必须支持：

- 用户列表
- 用户搜索与筛选
- 用户详情
- 认证审核
- 启用与禁用
- 异常账号复核

### 3.3 商品与审核管理

必须支持：

- 分类管理
- 商品列表
- 商品详情
- 商品上下架
- 资料类商品审核
- 审核通过与驳回

### 3.4 店铺与订单管理

必须支持：

- 店铺列表
- 店铺详情
- 店铺状态管理
- 订单列表
- 订单详情
- 支付状态查看
- 退款处理基础能力

### 3.5 举报与治理

必须支持：

- 举报列表
- 举报详情
- 举报处理结果
- 风控限制记录

## 4. 数据统计与洞察

详细要求见：

- [analytics-requirements.md](./analytics-requirements.md)
- [user-preferences-and-profile-insights.md](./user-preferences-and-profile-insights.md)

当前阶段要求如下。

### 4.1 用户侧统计

应支持：

- 累计消费金额
- 已购商品数
- 最近购买
- 最近浏览
- 收藏偏好
- 常购分类

### 4.2 店铺侧统计

应支持：

- 本月销售额
- 本月订单数
- 热销商品
- 浏览量
- 复购用户数

约束：

- 当前阶段允许先交付页面壳子、指标定义、接口预留
- 真实统计结果以持久化完成后的聚合查询为准
