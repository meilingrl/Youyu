# 课程约束汇总

## 目的

本文件只记录课程对项目的硬性要求，不负责业务场景设计。

## 小作业要求

### 作业 1

- 使用 `Vue` 实现客户管理功能页面
- 必须包含增删改查能力

### 作业 2

- 使用 `Vue` 实现登录验证功能
- 实现系统主画面

### 作业 3

- 使用智能体开发工具开发数据分析系统
- 需覆盖数据采集、数据清洗、数据挖掘、数据可视化全流程

## 大作业要求

- 开发基于 Web 的电商平台
- 包含客户端与管理后台
- 数据库使用 `MySQL`
- 技术要求覆盖 `Java`、`HTML`、`CSS`、`JavaScript`
- 后端要求覆盖 `Servlet`、`JSP`、`Listener`、`Filter`、`JDBC`
- 框架要求覆盖 `Spring`、`Vue3`、`MyBatis`

## 进阶要求

- 前端优先使用 `Element Plus`
- 前端状态管理使用 `Pinia`
- 需要实现认证鉴权功能
- 可选用 `OAuth` 或 `Shiro`
- 前台与后台界面需简洁美观、操作流畅、布局统一
- 需要响应式适配能力
- 需要图片上传、数据分页等能力

## 拓展方向

- `Redis` 缓存热点数据
- 架构拆分
- `Docker` 容器化部署
- 异常处理与容错
