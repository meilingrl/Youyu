# Resources

This directory stores non-code project resources.

- `references/`: research material and supporting references
- `assets/`: images, icons, brand assets, and design source material
- `reports/`: presentation and reporting material

## Brand Assets

Brand-related assets live under `assets/brand/`.

- `assets/brand/shared/`: reusable templates and shared references
- `assets/brand/youyu/`: the current `友余 / Youyu` brand workspace

Keep approved current assets in a `current/` folder and preserve generated or older variants in matching `archive/` folders for traceability.
