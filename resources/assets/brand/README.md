# Brand Assets

This directory is the home for visual identity assets used by the project.

## Layout

- `shared/`: cross-brand templates, moodboards, and reusable references
- `youyu/`: the active brand workspace for `友余 / Youyu`

## Rules

- Put the currently selected version of an asset in a `current/` folder.
- Keep earlier generated drafts in `archive/generated/`.
- Prefer descriptive English filenames with version suffixes such as `youyu-logo-bag-mark-v1.png`.
- Keep source exports, transparent variants, and cropped platform assets in separate folders instead of mixing everything into one directory.

## Suggested Growth

As the asset library grows, keep work grouped by usage:

- `logo/`: primary marks, alternates, lockups
- `wordmark/`: text-only brand marks
- `app-icon/`: square icons, launch icons, favicons
- `social/`: covers, avatars, share cards
- `templates/`: reusable poster or slide templates
