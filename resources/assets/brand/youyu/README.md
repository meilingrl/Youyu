# Youyu Brand Workspace

Brand name: `友余`

English name: `Youyu`

Core meaning:

- `友`: friendship, peer connection, warmth between students
- `余`: surplus books, goods, notes, and experience passed forward

Current slogan:

`学有余，爱有续`

## Directory Guide

- `logo/current/`: currently selected logo files
- `logo/archive/generated/`: generated logo history kept for reference
- `wordmark/`: text-only brand marks
- `app-icon/`: square icon variants for app and favicon use
- `social/`: social-media-ready assets and covers
