# Tests

This directory holds test assets that live outside the main source trees.

Currently the primary test suites are co-located with their source:

- **Backend unit/integration tests**: `backend/src/test/` — run with `mvnw.cmd test` (H2 in-memory, no MySQL needed)
- **Frontend unit tests**: `frontend/src/` (`**/__tests__/*.test.js`) — run with `npm test` from `frontend/`
- **E2E tests**: `frontend/tests/e2e/` — run with `npm run test:e2e` (requires backend + frontend running)

This directory is reserved for:
- Cross-cutting test data or fixtures that don't belong to a single subproject
- Acceptance records and manual test checklists
- Test reports generated by CI
